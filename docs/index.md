  - title: Precision Ratings
    details: Configurable step values (0.5, 0.25, 0.2, 0.1) with fractional visual rendering for half, quarter, and tenth stars.
  - title: RTL Support
    details: Automatic right-to-left behavior via the native HTML dir attribute. No extra configuration required.
  - title: Touch Support
    details: Tap to select and drag across stars to preview ratings on touch devices. RTL-aware.
  - title: Custom Icon Slots
    details: Replace the default star with any inline content — emoji, SVG, or custom HTML — via the filled and empty named slots.
  - title: Size Variants
    details: Named presets (sm, md, lg) scale both star size and gap. Pass a number for an exact pixel size.
  - title: Color Customization
    details: Control filled, empty, and hover colors via props. Override at the theme level with CSS custom properties.
  - title: SSR-Safe
    details: All browser APIs are properly guarded. Works in Nuxt 3 and Vite SSR environments without configuration.
  - title: TypeScript-First
    details: Strict types for props, events, and slots. Full IntelliSense support out of the box.
